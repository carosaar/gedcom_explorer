GEDCOM Explorer 1.3
===================

Vielen Dank für die Nutzung des GEDCOM Explorer!

Dieses Programm dient zur Analyse und Extraktion von GEDCOM-Daten
und ermöglicht den Export in strukturierte CSV-Dateien.

------------------------------------------------------------

Schnellstart
------------

1. Programm starten:
   ged_explorer.exe

2. GEDCOM-Datei auswählen

3. Hauptmerkmal (L1-TAG) festlegen

4. Untertags auswählen

5. Ausgabe als CSV starten

------------------------------------------------------------

Batch-Nutzung (ohne GUI)
------------------------

Das Programm kann auch direkt mit einer Definitionsdatei gestartet werden:

   ged_explorer.exe config.gexp

In diesem Fall wird die Verarbeitung automatisch ausgeführt,
ohne die grafische Oberfläche zu öffnen.

------------------------------------------------------------

Hinweis zu Windows-Sicherheit
----------------------------

Beim ersten Start kann Windows eine Sicherheitswarnung anzeigen:

„Der Computer wurde durch Windows geschützt“

In diesem Fall:
- „Weitere Informationen“ klicken
- „Trotzdem ausführen“ wählen
